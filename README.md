# Pet Cafe Oradea

A simple web app where you can see more about Pet Cafe Oradea.

## STRUCTURE 🖥️

- React JS Vite
- Tailwind CSS

# Features & Usage 🔥

- [x] Menu page
- [x] Gallery page

- [x] Social Media Integration
